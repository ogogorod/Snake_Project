# Snake Game with Pygame



## Features

- Snake movement
- Food spawning
- Collision detection
- Score
- Increasing speed
- Game over screen
- Restart option

## How to run

Install pygame:

```bash
pip install pygame
```

Run the game:

```bash
python main.py
```

## Controls

- Arrow keys: move the snake
- R: restart after game over
- Q or Escape: quit after game over
